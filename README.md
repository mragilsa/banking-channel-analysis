# Banking Channel Performance and Transaction Efficiency Analysis

This repository analyzes banking channel performance and transaction efficiency using the USA Banking Transactions Dataset. The project evaluates transaction volume, value, operational cost, fraud, and failure rates to provide actionable recommendations for optimizing banking channels.

## Dataset
Source: [Kaggle – USA Banking Transactions Dataset (2023-2024)](https://www.kaggle.com/datasets/pradeepkumar2424/usa-banking-transactions-dataset-2023-2024)

Dataset Description:
- Contains ~5,000 transaction records across Cash, Credit Card, Debit Card, E-Wallet, and Online Transfer
- Includes transaction amount, date, customer demographics, and transaction status
- Suitable for channel performance and efficiency analysis

## Conclusion
E-wallets are the most cost-efficient and high-value channel, focus on digital channels and improve debit card reliability to optimize overall transaction efficiency.
